# Prompt Library

This library now separates artifacts by how they are used:

## 1. Chat Prompts
Files in [chat-prompts](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/chat-prompts) are prompt artifacts you paste directly into a chat with an LLM.

- [activity-library.md](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/chat-prompts/activity-library.md): reusable classroom and worksheet activity prompts.
- [starter-prompt-toolkit.md](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/chat-prompts/starter-prompt-toolkit.md): reusable starter prompts for diagnostic tutoring conversations.

## 2. System Prompts
Files in [system-prompts](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/system-prompts) are behaviour prompts for custom GPTs or other system-prompt driven assistants.

- [process](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/system-prompts/process): helps a custom GPT act like a teacher guiding a student through a larger project or process.
- [understand](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/system-prompts/understand): helps a custom GPT teach and probe specific concepts, especially for exam-style or knowledge-heavy content.
- [grading](/home/craigjefferies/projects/ncea-tech-llm-artifacts/prompt-library/system-prompts/grading): analytic grading prompts aligned to the OMI `evidence_guide` structure.

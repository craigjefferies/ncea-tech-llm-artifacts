**Task:** Generate a short starter activity for retrieval and low-Bloom concept recall.

**Use this when:**  
- you want a warm-up at the start of a lesson  
- students need to retrieve prior knowledge before deeper work  
- you want quick participation and low writing load

**Inputs to use:**  
- a concept, key term, misconception, or tightly scoped OMI idea  
- optional standard / OMI rubric / content notes  
- optional year level

**Non-negotiables:**  
- Keep the cognitive demand mainly at recall or simple recognition.  
- Do not turn the starter into a full analysis or evaluation task.  
- Keep the tone light, inviting, and classroom-ready.  
- If an OMI or concept is not supplied, choose one clearly supported by the provided material.

**Allowed Activity Types:**  
Select **one** activity type that best fits the concept:
- Matching (concept <-> definition / example)
- Multiple Choice Question
- True / False
- Odd One Out with a very short reason
- Fill the Blank
- Spot the Output (short code / short process prediction)
- Name the Concept from an example

**Design Rules:**  
- The task should take about 2-5 minutes.  
- Use plain student-facing language.  
- Keep the whole activity compact.  
- For programming prompts, code snippets must be very short.  
- If helpful, include a one-line teacher answer key.

**Output Format:**  

## Starter Activity  
- **Title / Lead-in:**  
- **Instructions for students:**  
- **Activity content:**  

## Teacher Note  
- **Correct answer(s):**  
- **What this is checking:** one short line

**Goal:**  
Produce a quick classroom warm-up that builds recall, confidence, and readiness for deeper learning.

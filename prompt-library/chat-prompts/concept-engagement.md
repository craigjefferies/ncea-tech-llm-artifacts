**Task:** Create a concept-engagement mini-package that helps students understand, discuss, and apply one high-leverage idea from the supplied standard or OMI rubric.

**Use this when:**  
- you are introducing or revisiting an important concept  
- students need more than a definition  
- you want a short activity that builds understanding before formal assessment work

**Inputs to use:**  
- Achievement Standard text and/or OMI JSON  
- optional content knowledge notes  
- optional student age / year level / class context

**Concept Selection Rule:**  
- If a concept is explicitly supplied, use it.  
- If not, choose one concept that is clearly supported by the source material and important for success in the rubric.  
- Do not invent a concept just because it sounds curriculum-adjacent.

**Teaching Principles:**  
- Start concrete before abstract.  
- Connect the concept to function, consequence, or real use.  
- Prefer clarity over cleverness.  
- Make student thinking visible through talk, short reasoning, or choice.  
- Keep examples and dilemmas culturally respectful and NZ-relevant where possible.

**Design Rules:**  
- Create three engagement moves:
  1. **Explainer** — give three short explanation styles:
     - analogy
     - step-by-step
     - diagram-friendly description of what to visualise
  2. **Socratic Questions** — 4-5 questions that move from recall -> understanding -> application -> reasoning -> reflection
  3. **Discussion / Decision Prompt** — 1-2 short, open-ended real-world prompts that create useful reasoning, not empty opinion
- If the topic is genuinely debatable, use a dilemma.  
- If it is not naturally debatable, use a decision prompt, comparison, or “what would you do here?” scenario instead.  
- Keep each section compact and student-friendly.  
- Where relevant, quietly support likely OMI evidence without naming the schema.
